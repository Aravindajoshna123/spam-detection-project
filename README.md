# spam-detection-project
# 📩 Spam Detection using NLP

This project detects whether a message is Spam or Not Spam using Machine Learning and Natural Language Processing (NLP).

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn

## 📊 Model Used
- Multinomial Naive Bayes

## ⚙️ How it Works
1. Load dataset
2. Convert text into numerical form using CountVectorizer
3. Train model
4. Predict whether message is spam or not

## 🎯 Accuracy
Achieved accuracy: **98%**

## ▶️ How to Run
1. Install required libraries:
   pip install pandas scikit-learn

2. Run the Python file:
   python main.py

## 📌 Example
Input: "Free money offer"
Output: Spam
